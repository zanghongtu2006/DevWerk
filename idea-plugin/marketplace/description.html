DevWerk is an AI-powered tool window for IntelliJ-based IDEs that generates and applies real project changes via structured CodeOps (file operations), not just chat suggestions.

  • Generate real files and folders — create, update, or delete paths directly inside your project
  • CodeOps JSON workflow — models return a structured response (reply, code_tree, ops)
  • Multiple providers — works with local Ollama, OpenAI GPT, Google Gemini, plus optional dedicated backend integration
  • Runs locally — operations are executed in your IDE; API keys are stored locally

How to use

  1. Open the DevWerk tool window (right side of the IDE)
  2. Configure your AI provider via the ⚙ Settings button
  3. Describe what you want to build, then DevWerk applies the returned file operations

Privacy & Security
DevWerk does not upload your API keys. All file operations are executed locally inside your IDE.
